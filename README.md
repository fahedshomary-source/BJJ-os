# BJJ OS

Your personal BJJ lifestyle operating system. A progressive web app (PWA) for tracking training, hydration, nutrition timing, smoking awareness, and technique progress.

## Features

- Daily schedule adapted to BJJ / strength / rest day type
- Hydration tracker (250ml cups, add/remove)
- Smoking awareness log (intentional vs automatic)
- Technique library with real YouTube video thumbnails
- Drilling session generator (partner or solo, 10/20/30 min)
- Class log (what worked, what caught you)
- Push notifications for daily reminders
- Full week schedule view
- Stats and progress tracking
- Works offline after first load

## Install on iPhone

1. Open the app URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**
5. Open from home screen → go to Notifications tab → enable reminders

## Install on Android

1. Open in **Chrome**
2. Tap the **three dots** menu
3. Tap **Add to Home Screen**
4. Confirm

## Deploy your own copy

1. Fork this repo
2. Go to Settings → Pages → Source → GitHub Actions
3. Push any change to `main`
4. Your app will be live at `https://yourusername.github.io/bjj-os`

## Data

All data is stored locally on your device using localStorage. Nothing is sent to any server. Each user's data is completely separate.
